🐘 Informasi PHP

Tips Penggunaan PHP
📌 Cara menjalankan file PHP:
1. Simpan file dengan ekstensi .php
2. Letakkan di folder htdocs (XAMPP) atau www (WAMP)
3. Akses melalui browser: http://localhost/nama_file.php
4. Pastikan server web (Apache) sudah berjalan

📌 Struktur dasar PHP:
<?php
    // Kode PHP di sini
    echo "Hello World!";
?>

📌 Fungsi PHP yang sering digunakan:
• echo / print - Menampilkan output
• isset() - Memeriksa apakah variabel ada
• empty() - Memeriksa apakah variabel kosong
• include() / require() - Menyertakan file lain
• $_GET / $_POST - Mengambil data form
• mysqli_connect() - Koneksi ke database MySQL

Shortcut Keyboard:
• Ctrl + S = Download file
• Ctrl + D = Hapus semua kode
• Ctrl + Shift + P = Simpan sebagai PHP
